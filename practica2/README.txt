Para correr el archivo practica2_p2.py es necesario tener el archivo input en .txt y el archivo practica1.py 
en el mismo directorio a la hora de ejecutarlo. Y se ejecuta de la siguiente forma:

Windows:
C:\carpeta\del\proyecto> practica2_p2.py nombre_archivo_input.txt (numero de iteraciones M)

Linux/ MacOs
user@user-pc:~/carpeta/del/proyecto$ python3 practica2_p2.py nombre_archivo_input.txt (numero de iteraciones M)
